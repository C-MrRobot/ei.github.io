Hexo部署在GitHub Pages指导

1:本地安装Hexo环境
#apt install nodejs-lts
#npm install -g hexo-cli
#npm install hexo-deployer-git
#npm g
#npm d

2:本地准备部署工具
#apk install openssh git vim

3:生成ssh公钥
#ssh-keygen -t rsa -C “user.email”

4:Github环境准备
a.设置中先添加ssh公钥
b.创建github仓库 命名:username.github.io
c.git clone 仓库到本地目录
d.复制本地blog目录下的文件到仓库中
e.修改:’_config.yml’ 文件，末尾添加如下信息:
“deploy:
type: git
repo: git@github.com:C-MrRobot/ei.github.io.git
branch: main “ 注意:github默认是main分支
f.在clone下来的仓库中添加插件，配置githubconfig
#npm install hexo-deployer-git
#git config –global core.autocrlf false
#git config –global user.email “useremail”
#git config –global user.name “username”
g.推送代码到github
#hexo g & hexo d
h.前往github 仓库中的设置配置git pages页面，选择要显示的分支保存即可
注意:仓库属性为public 公开的才能提供页面服务，private 私有仓库不能显示，每改变一次仓库属性，需要重新配置git pages。
